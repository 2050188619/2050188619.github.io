JAVA中&&和&、||和|的区别

· 2021-04-06 · 
if(a==1 && b==2) 这是说既要满足a=1,也要满足b=2;
if(a==1 || b==2) 这是说只需要满足a=1,或者满足b=2;

而a & b和 a | b是二进制的与或运算
&同为1时为1，否则为0；
| 同为0时为0，否则为1；

短路求值

        int a=10,b=10;
        if((a=0)==0 || (b=20)==20){
        }
        System.out.println(b);
        if((a=0)==0 | (b=20)==20){
        }
        System.out.println(b);

为什么第一个输出的b没有改变，而第二个b改变了呢？
正是因为短路求值的原因，在 || 之前已经为true时，那么后面将会进行短路，不再继续操作。
而 | 却不会进行短路。
